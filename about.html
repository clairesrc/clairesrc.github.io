About

Software engineer,
systems generalist.

Architect, platform builder, infrastructure engineer, and technical leader with over 15 years of experience spanning custom kernel patchsets to distributed systems, production operations to developer experience.

I'm based in Chicago, Illinois, and I've spent the last 15+ years working across the stack: from PHP and shell scripts in the early days to distributed systems, platform engineering, and architecture at scale. I've run production infrastructure handling 500k+ requests per second, built developer platforms from scratch, and led engineering teams through critical infrastructure transitions. Language-agnostic by nature, lately I mostly write Go and Python.

I hold an AWS Certified Solutions Architect credential and have shipped systems across infrastructure, backend services, platform tooling, data pipelines, and observability, whatever the problem demands. I work deep when I need to (kernel parameters, GPU passthrough, audio latency) and wide when the architecture calls for it.

How I work

I believe the best systems are invisible: infrastructure that doesn't need attention, tooling that accelerates without getting in the way, architecture that absorbs complexity. I'm drawn to problems that span traditional boundaries: reliability and usability, performance and maintainability, depth and breadth.

I run Linux on all my machines, currently NixOS: its declarative model matches how I think about system configuration. My home network doubles as a lab for infrastructure experiments, GPU passthrough, container orchestration, and whatever else catches my interest.
